Welcome to the Disting mk4 Quick Guide. This guide is a cheat sheet for the 97 algorithms included with the Expert Sleepers Disting mk4 (as of firmware update 4.11). Since the bulk of the text of the manual is not repeated here, the Quick Guide is not designed to be your primary source for getting to know your Disting. Once you have a basic understanding of how the algorithms work, however, I hope that you’ll find this page a handy map to the ins, outs and parameter settings of each algorithm.

In addition to distilling the info to the basics, this guide is designed to be mobile-friendly, so you can view it without being tied to your computer. You can save the guide to the homescreen of your mobile device for easy mobile access.

If you find any errors or have any suggestions, please don’t hesitate to send them to damon@rustleworks.com. The code is open sourced under the MIT License. If you’d like to clone it for your own purposes, you can find the source at Github.

Happy Bleeping!
